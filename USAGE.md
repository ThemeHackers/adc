# คู่มือการใช้งานระบบจำลอง ADC (ADC Simulation System)

## ภาพรวมระบบ

ระบบจำลอง ADC (Artificial Dynamic Compaction) เป็นระบบติดตามและจำลองการบดอัดดินแบบไดนามิกโดยใช้หลักฟิสิกส์ที่ถูกต้อง ระบบนี้ออกแบบมาเพื่อการศึกษาและทดสอบการทำงานของระบบบดอัดดินแบบไดนามิกก่อนนำไปใช้งานจริง

## สูตรทางคณิตศาสตร์ที่ใช้ในระบบ

ระบบใช้สูตรทางฟิสิกส์ที่ถูกต้องดังนี้:

1. **พลังงานศักย์ (Potential Energy)**: Ep = m × g × h (จูล)
2. **พลังงานจลน์ (Kinetic Energy)**: Ek = ½ × m × v² (จูล)
3. **กำลังมอเตอร์ (Motor Power)**: P = F × v, โดย F = m × g (วัตต์)
4. **กำลังเครื่องกำเนิดไฟ (Generator Power)**: P_gen = (dEp/dt) × η (วัตต์)
5. **การตกอิสระ (Free Fall)**: v = v₀ + g × t, h = h₀ + v × t
6. **สูตร Menard สำหรับความลึกการบดอัด**: Z = n × √(M × H)
7. **การบดอัดดิน**: อิงจากพลังงานการกระแทกและคุณสมบัติของดิน
8. **แบตเตอรี่**: I = P / V, ความจุ = ∫I dt

## การติดตั้งระบบ

### ข้อกำหนดเบื้องต้น (Prerequisites)

- Node.js (เวอร์ชัน 18 หรือใหม่กว่า)
- npm หรือ yarn
- เบราว์เซอร์ที่รองรับ (Chrome, Firefox, Edge, Safari)

### ขั้นตอนการติดตั้ง

1. ดาวน์โหลดโค้ดจาก repository
2. เปิด Terminal และไปยังโฟลเดอร์โปรเจกต์
3. ติดตั้ง dependencies:
```bash
npm install
```
4. เริ่มระบบ:
```bash
npm run dev
```
5. เปิดเบราว์เซอร์และไปที่ http://localhost:3000

## การใช้งานระบบ

### 1. การเริ่มต้นระบบ

เมื่อเปิดระบบครั้งแรก คุณจะเห็นหน้าจอหลักที่แสดง:
- แผงควบคุม (Control Panel)
- การแสดงผลสด (Live Visualization)
- กราฟและตัวชี้วัดต่างๆ
- สถานะระบบ

### 2. การควบคุมระบบ

**ปุ่มควบคุมหลัก:**
- **Start**: เริ่มการจำลอง
- **Pause**: หยุดการจำลองชั่วคราว
- **Reset**: รีเซ็ตระบบกลับสู่ค่าเริ่มต้น

**การเปลี่ยนสถานะ:**
ระบบมี 4 สถานะหลัก:
- **IDLE**: รอการทำงาน
- **CHARGING**: กำลังชาร์จ (ยกลูกตุ้มขึ้นโดยใช้พลังงานแสงอาทิตย์)
- **DISCHARGING**: กำลังจ่ายไฟ (ลูกตุ้มลงช้าๆ เพื่อสร้างพลังงาน)
- **IMPACT**: การกระแทกดิน (ลูกตุ้มตกอิสระเพื่อบดอัดดิน)

### 3. การตั้งค่าระบบ (Configuration)

คลิกที่ไอคอน **Settings** (รูปฟันเฟือง) ที่มุมขวาบนเพื่อเปิดแผงตั้งค่า

**การตั้งค่าลูกตุ้ม (Pendulum Settings):**
- **Pendulum Mass (kg)**: มวลลูกตุ้ม (ค่าเริ่มต้น: 500 kg)
- **Max Height (m)**: ความสูงสูงสุด (ค่าเริ่มต้น: 15 m)
- **Gravity (m/s²)**: แรงโน้มถ่วง (ค่าเริ่มต้น: 9.81 m/s²)
- **Initial Soil Density (kg/m³)**: ความหนาแน่นดินเริ่มต้น (ค่าเริ่มต้น: 1600 kg/m³)

**การตั้งค่าระบบพลังงาน (Power System):**
- **Solar Power (W)**: พลังงานแสงอาทิตย์ (ค่าเริ่มต้น: 650 W)
- **Motor Power (W)**: พลังงานมอเตอร์ (ค่าเริ่มต้น: 750 W)
- **Motor Efficiency (%)**: ประสิทธิภาพมอเตอร์ (ค่าเริ่มต้น: 85%)
- **Generator Efficiency (%)**: ประสิทธิภาพเครื่องกำเนิดไฟ (ค่าเริ่มต้น: 90%)

**การตั้งค่าแบตเตอรี่ (Battery System):**
- **Battery Voltage (V)**: แรงดันแบตเตอรี่ (24V หรือ 48V)
- **Initial Capacity (%)**: ความจุเริ่มต้น (ค่าเริ่มต้น: 50%)

หลังจากตั้งค่าแล้ว คลิก **Save Configuration** เพื่อบันทึก หรือ **Reset to Default** เพื่อกลับสู่ค่าเริ่มต้น

### 4. การใช้สถานการณ์จำลอง (Demo Scenarios)

ระบบมีสถานการณ์จำลองที่ตั้งไว้ล่วงหน้า:

- **Quick Demo**: การสาธิตแบบรวดเร็ว (30 วินาที)
- **Full Cycle**: วงจรเต็ม (60 วินาที)
- **Stress Test**: ทดสอบความเครียด (45 วินาที)
- **Custom**: ใช้การตั้งค่าปัจจุบัน

เลือกสถานการณ์ที่ต้องการและคลิกเพื่อเริ่ม

### 5. การอ่านข้อมูลจากระบบ

**ตัวชี้วัดหลัก:**
- **Solar Input**: พลังงานแสงอาทิตย์ที่รับเข้ามา (วัตต์)
- **Motor Power**: กำลังมอเตอร์ (วัตต์)
- **Generator**: กำลังเครื่องกำเนิดไฟ (วัตต์)
- **Load Power**: กำลังโหลด (วัตต์)
- **Battery Capacity**: ความจุแบตเตอรี่ (%)
- **Battery Current**: กระแสแบตเตอรี่ (แอมแปร์)

**พลังงาน:**
- **Potential Energy**: พลังงานศักย์ (กิโลจูล)
- **Kinetic Energy**: พลังงานจลน์ (กิโลจูล)
- **Total Energy**: พลังงานรวม (กิโลจูล)

**ข้อมูลดิน:**
- **Soil Density**: ความหนาแน่นดิน (kg/m³)
- **Soil Compaction**: การบดอัดดิน (%)
- **Total Impacts**: จำนวนการกระแทกทั้งหมด

### 6. กราฟและการแสดงผล

**กราฟแบบเรียลไทม์:**
- **Total Energy**: แสดงพลังงานรวมตามเวลา
- **Generator Power**: แสดงกำลังเครื่องกำเนิดไฟตามเวลา
- **Pendulum Height**: แสดงความสูงลูกตุ้มตามเวลา

**การแสดงผลสด:**
- แสดงลูกตุ้มและหลุมดินแบบเรียลไทม์
- สีของลูกตุ้มเปลี่ยนตามสถานะ:
  - เขียว: CHARGING
  - ฟ้า: DISCHARGING
  - แดง: IMPACT
  - เทา: IDLE

### 7. การส่งออกข้อมูล (Data Export)

**ส่งออกข้อมูลเป็น CSV:**
1. คลิกที่ปุ่ม **Export CSV**
2. ไฟล์จะถูกดาวน์โหลดอัตโนมัติ

**ส่งออกข้อมูลเป็น JSON:**
1. คลิกที่ปุ่ม **Export JSON**
2. ไฟล์จะถูกดาวน์โหลดอัตโนมัติ

**ดาวน์โหลดรายงาน:**
1. คลิกที่ปุ่ม **Download Report**
2. ไฟล์ HTML รายงานจะถูกดาวน์โหลด

### 8. การบันทึกข้อมูลลงประวัติ (History)

1. คลิกที่ปุ่ม **Save Current Run to History**
2. ข้อมูลการจำลองปัจจุบันจะถูกบันทึก
3. สามารถดูประวัติได้จากแผงประวัติ

### 9. การตั้งค่าการแจ้งเตือน (Alerts)

ระบบจะแจ้งเตือนอัตโนมัติเมื่อ:
- แบตเตอรี่ต่ำกว่า 20%
- ลูกตุ้มถึงความสูงสูงสุด
- การบดอัดดินเสร็จสิ้น
- การชาร์จเสร็จสิ้น
- การจ่ายไฟเสร็จสิ้น

## การแก้ไขปัญหา (Troubleshooting)

### ระบบไม่ทำงาน
- ตรวจสอบว่าได้ติดตั้ง dependencies ทั้งหมด
- ลองรีสตาร์ทเซิร์ฟเวอร์ด้วย `npm run dev`
- ตรวจสอบ Console ในเบราว์เซอร์สำหรับข้อผิดพลาด

### กราฟไม่แสดงข้อมูล
- ตรวจสอบว่าได้กดปุ่ม Start แล้ว
- ตรวจสอบว่าระบบอยู่ในสถานะที่ทำงานได้

### ข้อมูลแสดงเป็น NaN
- ตรวจสอบการตั้งค่าใน Configuration Panel
- ตรวจสอบว่าค่าทั้งหมดเป็นตัวเลขที่ถูกต้อง

## ข้อจำกัดของระบบ

- ระบบเป็นการจำลองเท่านั้น ไม่ใช่การทำงานจริง
- ค่าที่แสดงเป็นค่าจำลองตามสูตรฟิสิกส์
- การบดอัดดินเป็นแบบจำลองอย่างง่าย
- ข้อมูลที่แสดงไม่ใช่ข้อมูลจริงจากฮาร์ดแวร์

## การนำไปใช้กับฮาร์ดแวร์จริง

หากต้องการใช้ระบบกับฮาร์ดแวร์จริง จำเป็นต้อง:

1. เชื่อมต่อเซนเซอร์ต่างๆ (เซนเซอร์ความสูง, เซนเซอร์ความเร็ว, เซนเซอร์แรงดัน, เซนเซอร์ดิน)
2. ใช้ไมโครคอนโทรลเลอร์ (Arduino, ESP32, Raspberry Pi) เพื่ออ่านข้อมูล
3. ส่งข้อมูลผ่าน Serial, USB, WiFi, หรือ Ethernet
4. สร้าง Backend Server เพื่อรับและประมวลผลข้อมูล
5. แก้ไขระบบจำลองให้รับข้อมูลจากฮาร์ดแวร์แทนการจำลอง
