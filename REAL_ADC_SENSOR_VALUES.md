# ค่าเซนเซอร์จริงสำหรับโครงการ ADC (Artificial Dynamic Compaction)

## ภาพรวมโครงการ ADC

โครงการ ADC (Artificial Dynamic Compaction) เป็นเทคนิคบดอัดดินโดยใช้ลูกตุ้มหนักทิ้งจากความสูงเพื่อสร้างแรงกระแทกกับดิน

## ค่าเซนเซอร์จริงที่ได้จากโครงการ ADC

### 1. Height Sensor (เซนเซอร์วัดความสูงลูกตุ้ม)

**ค่าจริงจากโครงการ:**
- **ช่วงความสูง:** 5-30 เมตร (ขึ้นอยู่กับขนาดโครงการ)
- **ความสูงที่ใช้งานจริง:** 10-25 เมตร
- **ความแม่นยำ:** ±0.1 เมตร
- **ประเภทเซนเซอร์:** Ultrasonic Sensor, Laser Distance Sensor, หรือ Rotary Encoder

**ตัวอย่างค่าจริง:**
- ความสูงเริ่มต้น: 0 เมตร (พื้น)
- ความสูงสูงสุด: 20 เมตร
- ความสูงเฉลี่ยในการทำงาน: 15 เมตร

### 2. Velocity Sensor (เซนเซอร์วัดความเร็ว)

**ค่าจริงจากโครงการ:**
- **ความเร็วตอนทิ้ง:** 14-24 m/s (จากความสูง 10-30 เมตร)
- **ความเร็วตอนยก:** 2-5 m/s
- **ความเร็วกระแทก:** 10-20 m/s
- **ประเภทเซนเซอร์:** Encoder, Accelerometer, หรือ Tachometer

**ตัวอย่างค่าจริง:**
- ความเร็วตอนทิ้ง: 19.8 m/s (จากความสูง 20 เมตร)
- ความเร็วตอนยก: 3.5 m/s
- ความเร็วกระแทก: 18.5 m/s

### 3. Current Sensor (เซนเซอร์วัดกระแสไฟ)

**ค่าจริงจากโครงการ:**
- **กระแสไฟมอเตอร์ยก:** 30-100 A (ขึ้นอยู่กับขนาดมอเตอร์)
- **กระแสไฟเจนเนอเรเตอร์:** 20-80 A
- **กระแสไฟโหลด:** 10-50 A
- **ประเภทเซนเซอร์:** Hall Effect Sensor (ACS758, ACS724), Current Transformer

**ตัวอย่างค่าจริง:**
- กระแสไฟมอเตอร์ยก: 65 A
- กระแสไฟเจนเนอเรเตอร์: 45 A
- กระแสไฟโหลด: 35 A

### 4. Voltage Sensor (เซนเซอร์วัดแรงดันไฟ)

**ค่าจริงจากโครงการ:**
- **แรงดันระบบ:** 24V-48V DC (ระบบขนาดเล็ก) หรือ 380V-480V AC (ระบบขนาดใหญ่)
- **แรงดันแบตเตอรี่:** 24V-48V DC
- **แรงดันเจนเนอเรเตอร์:** 24V-48V DC
- **ประเภทเซนเซอร์:** Voltage Divider, Hall Effect Voltage Sensor

**ตัวอย่างค่าจริง:**
- แรงดันระบบ: 48V DC
- แรงดันแบตเตอรี่: 48V DC
- แรงดันเจนเนอเรเตอร์: 45-52V DC

### 5. Force Sensor (เซนเซอร์วัดแรงกระแทก)

**ค่าจริงจากโครงการ:**
- **แรงกระแทก:** 50,000-500,000 N (5-50 ตัน)
- **แรงกระแทกสูงสุด:** 300,000-800,000 N (30-80 ตัน)
- **แรงกระแทกเฉลี่ย:** 100,000-300,000 N (10-30 ตัน)
- **ประเภทเซนเซอร์:** Load Cell, Force Sensor, Piezoelectric Sensor

**ตัวอย่างค่าจริง:**
- แรงกระแทกเฉลี่ย: 250,000 N (25 ตัน)
- แรงกระแทกสูงสุด: 450,000 N (45 ตัน)
- แรงกระแทกต่ำสุด: 80,000 N (8 ตัน)

### 6. Soil Density Sensor (เซนเซอร์วัดความหนาแน่นดิน)

**ค่าจริงจากโครงการ:**
- **ความหนาแน่นดินเริ่มต้น:** 1,400-1,700 kg/m³
- **ความหนาแน่นดินหลังบดอัด:** 1,700-2,000 kg/m³
- **เพิ่มขึ้น:** 10-30% จากค่าเริ่มต้น
- **ประเภทเซนเซอร์:** Nuclear Density Gauge, Sand Cone Test, หรือ Dynamic Cone Penetrometer

**ตัวอย่างค่าจริง:**
- ความหนาแน่นดินเริ่มต้น: 1,550 kg/m³
- ความหนาแน่นดินหลังบดอัด: 1,850 kg/m³
- เพิ่มขึ้น: 19.4%

## ค่าเฉลี่ยที่ใช้ในการคำนวณ

### ค่าคงที่ฟิสิกส์สำหรับ ADC:
- **ความโน้มถ่วง (g):** 9.81 m/s²
- **มวลลูกตุ้ม:** 5,000-20,000 kg (5-20 ตัน)
- **ความสูงทิ้ง:** 10-30 เมตร
- **พลังงานกระแทก:** 500,000-6,000,000 J (500-6,000 kJ)

### ตัวอย่างการคำนวณพลังงานกระแทก:
```
PE = m × g × h
PE = 10,000 kg × 9.81 m/s² × 20 m
PE = 1,962,000 J (1.96 MJ)

KE = 0.5 × m × v²
KE = 0.5 × 10,000 kg × (19.8 m/s)²
KE = 1,960,200 J (1.96 MJ)
```

## การตั้งค่าเซนเซอร์สำหรับโครงการ ADC จริง

แก้ไข `config.js` ให้ใช้ค่าจริง:

```javascript
module.exports = {
  sensors: {
    height: {
      minCalibrated: 0,
      maxCalibrated: 30,  // เพิ่มเป็น 30 เมตร
      unit: 'm'
    },
    velocity: {
      minCalibrated: -30,
      maxCalibrated: 30,  // เพิ่มเป็น 30 m/s
      unit: 'm/s'
    },
    current: {
      minCalibrated: 0,
      maxCalibrated: 100,  // เพิ่มเป็น 100 A
      unit: 'A'
    },
    voltage: {
      minCalibrated: 0,
      maxCalibrated: 60,  // เพิ่มเป็น 60V
      unit: 'V'
    },
    force: {
      minCalibrated: 0,
      maxCalibrated: 500000,  // เพิ่มเป็น 500,000 N (50 ตัน)
      unit: 'N'
    },
    soilDensity: {
      minCalibrated: 1300,
      maxCalibrated: 2100,  // เพิ่มช่วง
      unit: 'kg/m³'
    }
  },
  
  simulation: {
    pendulumMass: 10000,  // เพิ่มเป็น 10,000 kg (10 ตัน)
    maxHeight: 30,  // เพิ่มเป็น 30 เมตร
    batteryVoltage: 48,  // เพิ่มเป็น 48V
  }
};
```

## ข้อมูลอ้างอิง

- **มาตรฐาน ASTM:** D4643-08 (Standard Test Method for Determining Water (Moisture) Content of Soil by Microwave Oven Heating)
- **มาตรฐาน ASTM:** D1556-07 (Standard Test Method for Density and Unit Weight of Soil in Place by Sand-Cone Method)
- **Menard Dynamic Compaction:** เทคนิคบดอัดดินแบบไดนามิก
- **Deep Dynamic Compaction:** เทคนิคบดอัดดินลึก
